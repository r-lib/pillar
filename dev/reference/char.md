# Format a character vector in a tibble

These functions are reexported as
[`tibble::char()`](https://tibble.tidyverse.org/reference/char.html) and
[`tibble::set_char_opts()`](https://tibble.tidyverse.org/reference/char.html).

## Usage

``` r
char(
  x,
  ...,
  min_chars = NULL,
  shorten = c("back", "front", "mid", "abbreviate")
)

set_char_opts(
  x,
  ...,
  min_chars = NULL,
  shorten = c("back", "front", "mid", "abbreviate")
)
```
