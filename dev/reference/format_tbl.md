# Formatting of tbl objects

See
[tibble::formatting](https://tibble.tidyverse.org/reference/formatting.html)
for details.

## Usage

``` r
# S3 method for class 'tbl'
print(
  x,
  width = NULL,
  ...,
  n = NULL,
  max_extra_cols = NULL,
  max_footer_lines = NULL
)

# S3 method for class 'tbl'
format(
  x,
  width = NULL,
  ...,
  n = NULL,
  max_extra_cols = NULL,
  max_footer_lines = NULL
)
```
